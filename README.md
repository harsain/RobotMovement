#Robot Movement

## Commands:

- PLACE
  (requires 3 arguments- Int x, Int y, String direction) 
- REPORT
- LEFT
- RIGHT
- MOVE
- QUIT

The first command has to be PLACE / QUIT
command cases are ignored so both uppercase and lowercase can be used
